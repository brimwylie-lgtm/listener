How to capture a webinar or podcast replay
  1. Open the video/audio in a separate Chrome tab
  2. Come back to this tab and choose Browser tab audio
  3. Click Start recording
  4. Chrome will ask you to choose a tab to share — pick the tab playing the content
  5. Make sure to check "Share tab audio" in the Chrome share dialog
  6. Hit play on the video — playback speed doesn't matter, audio is captured as-is
  7. When done, click Stop recording, then transcribe
Whisper API setup
  1. Go to platform.openai.com/api-keys
  2. Create an API key and paste it in the field top-right
  3. Cost: roughly $0.006 per minute of audio — a 60-min webinar costs ~$0.36
  4. Your key is never stored — it lives only in this browser session
Speaker identification

After transcription, Whisper returns timestamped segments. This tool groups those into speaker blocks based on pauses and voice changes. Click any speaker badge in the transcript to rename them. Use Label speakers in the toolbar to set all names at once.
