How to use
1
Select Tab for webinars or replays — open the video in another Chrome tab first
2
Hit Start recording, then pick the tab in Chrome's share dialog. Check "Share tab audio"
3
When done, hit Stop and download the file
4
Drag the file into Claude and ask for a transcript — works at any playback speed
